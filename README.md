# DR-MDP

Reinforcement learning research codebase for **Delayed, Aggregate, and Anonymous Feedback (DAAF)** in MDPs.

## Env

```sh
uv venv --python 3.12
# will run uv sync
make pip-sync
```

## Run Tests

```sh
make test
```

## DFDRL: Deep RL for Delayed Feedback

Reward estimation for policy control, in off-policy deep reinforcement learning.
